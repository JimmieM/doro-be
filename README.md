## Description

Doro SR backend project.

[Click here](https://github.com/JimmieM/doro-fe) to see full documentation of this Full-stack project

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
